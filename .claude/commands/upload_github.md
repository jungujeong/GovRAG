github에 파일 올리기
1. 새로운 브랜치 생성
2. github에 commit 하고 push 하기
3. Issue 만들기
4. PR 하기 (develop 브랜치와 merge)
- 앞서 만든 Issue와 연동하기
5. develop 브랜치와 merge하기
6. 앞서만든 필요없는 branch 로컬 및 github에서 삭제

주의 사항:
1. commit, Issue, PR 등에서는 반드시 CLAUDE 관련된 태그를 함께하면 안됩니다. commit 포함
2. commit title은 반드시 영어, 설명은 영어로 해야합니다.
3. Issue와 PR의 title은 반드시 한국어, 설명도 한국어로 해야합니다.
4. 지금까지 작업한 내용을 종합하여 github에 올리기

Issue 양식:
### 📌 작업 개요
- 해당 기능이 어떤 목적을 가지고 있는지 간략하게 설명해주세요.

### 📝 작업 내용
- [ ] 해야 할 작업 1
- [ ] 해야 할 작업 2
- [ ] 해야 할 작업 3

### 💡 참고 사항
- 참고할 자료, 관련된 문서, 기타 공유하고 싶은 내용이 있다면 작성해주세요.
